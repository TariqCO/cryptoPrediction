# Crypto Prediction Platform (Predicta)

A **web-based cryptocurrency prediction platform** that allows users to submit their market predictions, track cryptocurrency trends in real-time, and get AI-powered insights. Built with a modern tech stack, the platform combines user-driven predictions with **Gemini AI integration** to provide accurate verdicts and summaries.

## Key Features

* **User Authentication**
  Secure registration and login system with JWT-based authentication to protect user data and prediction history.

* **Prediction Submission**
  Users can submit predictions for various cryptocurrencies including target price, direction (up/down), and fulfillment time.

* **AI-Powered Analysis**
  Integrates **Gemini AI** to analyze all user predictions and generate comprehensive summaries, common trends, and actionable verdicts.

* **Live Cryptocurrency Rates**
  Real-time tracking of cryptocurrency prices with automatic updates, helping users make informed predictions.

* **Responsive & Modern UI**
  Built with React, Tailwind CSS, and ShadCN UI components for a sleek and user-friendly interface on all devices.

## Tech Stack

* **Frontend:** React, Tailwind CSS, ShadCN UI, Framer Motion
* **Backend:** Node.js, Express
* **Database:** MongoDB
* **AI Integration:** Gemini AI API
* **Authentication:** JWT

## Why This Platform Stands Out

* Combines **community-driven predictions** with **AI-powered analysis**.
* Tracks **live crypto market trends** for accurate decision-making.
* Easy-to-use interface with a **professional and interactive dashboard**.
